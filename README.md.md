# What is this?
A simple guide/instruction to creating a NAS server using OMV (OpenMediaVault) in RPi devices.

The article doesn't aim to be a complete documentation with a detailed investigation of each point in OpenMediaVault, instead there's a solution of making a NAS server for daily usage only (e.g. disks configuration, users management...).

# Preparation
* Raspberry Pi (2B and newer, >1gb of RAM) and its accessories (for more details [click here](https://github.com/EdwardBorry/Raspberry-Pi-connection)).
* Operating system as a Raspberry Pi OS Lite in your storage device.
* Extra PC (or another device) for SSH connecting.
